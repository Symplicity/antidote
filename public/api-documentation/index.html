Users 

This endpoint is to manage users in the application

User List 

A list of users

Get users
GET
/users{?name,sort,limit}

Get a list of users. Example:

https://entic.com/api/users?sort=name&limit=5
  • Parameters
  • name
    string (optional) Example: alice

    Search for a user by name

    sort
    string (optional) Default: name Example: joined

    Which field to sort by

    Choices: name

    limit
    integer (optional) Default: 10 Example: 25

    The maximum number of users to return, up to 50

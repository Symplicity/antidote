Drugs 

With the drugs API you can retrieve information and reviews on individual drugs as well as list all drugs.

Drugs 

List
GET
/drugs{?keywords,alpha,page,limit}

Returns a list of all drugs. The drugs are returned in alphabetically sorted order. Example:

https://antidote.symplicity-opensource.com/api/drugs?keywords=Tylenol
  • Parameters
  • keywords
    string (optional) Example: Tylenol

    Search for a drug by keywords in label or description

    alpha
    string (optional) Example: a

    Search for a drug by first letter of the label

    page
    integer (optional) Example: 1

    Page number of drugs

    limit
    integer (optional) Default: 15 Example: 25

    The maximum number of drugs to return, up to 500

    limit

Create New
POST
/drugs/{id}/reviews{?keywords,page,limit}

Create a new drug review

  • Parameters
  • rating
    integer (required) Example: 3

    The rating the user gave the drug either 1, 2, 3 - 1 is best

    is_covered_by_insurance
    integer (required) Example: 1

    Was the drug covered by insurance?

    comment
    text (optional) Example: My pain has subsided considerably since I started taking this.

    User comments

    side_effects
    array (optional) Example: [1,2,3]

    Side effects experienced by the user

  • RequestShowHide
  • Headers
    Content-Type: application/json
    Body
    {
    "rating": "3",
    "is_covered_by_insurance": "1",
    "comment": "My pain has subsided considerably since I started taking this.",
    "side_effects": [1,2,3]
}
  • Response  201ShowHide
  • Body
    {
    "rating": "3",

    "id": 1515
}

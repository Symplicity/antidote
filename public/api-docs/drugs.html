API Documentation

Drugs 

With the drugs API you can retrieve information and reviews on individual drugs as well as list all drugs.

Drugs 

List
GET
/drugs

Returns a list of all drugs. The drugs are returned in alphabetically sorted order. Example:

https://antidote.symplicity-opensource.com/api/drugs?label=Tylenol
  • Parameters
  • label
    string (optional) Example: Tylenol

    Search for a drug by label

    limit
    integer (optional) Default: 10 Example: 25

    The maximum number of drugs to return, up to 50

  • Response  200ShowHide
  • Body
    {
    "data": [
        {
            "id": 1,
            "label": "Tylenol",
            "description": "Tylenol is used to treat pain."
        },
        {
            "id": 2,
            "label": "Claritin",
            "description": "Claritin is used to treat allergies"
        }
    ]
}
